# Чек-лист MVP-готовности

1) /api/health → ok
2) /api/search с фиктивными данными выдаёт 5/5 карточек ≤ 2с
3) Бот: /start, «Где поесть», «Куда сходить», «Опрос», «Добавить событие» работают
4) Краулеры кладут события (≥50 карточек по Севастополю)
5) Дедуп < 3%/день; валидный маршрут у ≥95% карточек
6) Админка: approve UGC, поднять рекламный слот
7) Логи в JSON, ошибки уходят в Sentry
8) Docker up на чистой машине запускается с нуля
