# ✅ Чеклист развертывания cudaCrimea

## 🔧 Подготовка сервера

- [ ] Установлен Ubuntu 20.04+ / Debian 11+ / CentOS 8+
- [ ] Минимум 2GB RAM (рекомендуется 4GB+)
- [ ] Минимум 10GB дискового пространства
- [ ] Установлен Docker 20.10+
- [ ] Установлен Docker Compose 2.0+
- [ ] Установлен Git
- [ ] Пользователь добавлен в группу docker
- [ ] Открыты порты: 80, 443 (и 22 для SSH)

## 📥 Клонирование и настройка

  - [ ] Разрешен порт 80 (HTTP)
  - [ ] Разрешен порт 443 (HTTPS)

- [ ] В `docker-compose.yml` закомментированы публичные порты db и redis
